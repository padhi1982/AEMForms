🤖 AEM Forms Agent Workflow

PDF to JSON/HTML Form Conversion with AI

📄

PDF Processing

Extract form structure from multi-page PDF documents using advanced text parsing.

🤖

AI-Powered

Google Gemini LLM analyzes form fields and generates structured JSON data.

🎨

Form Generation

Automatically generate HTML forms and AEM component specifications.

🔄 How It Works

PDF Upload → Agent 1: Parser → Agent 2: Builder → Agent 3: Mapper → Output

Agent 1 (Parser): Extracts form structure from PDF and generates JSON schema

Agent 2 (Builder): Converts JSON schema to interactive HTML form

Agent 3 (Mapper): Maps form fields to AEM component specifications

🚀 Quick Start

Installation

# Clone the repository git clone https://github.com/padhi1982/AEMforms.git cd AEMforms # Install dependencies pip install -r requirements.txt # Set up Gemini API key python3 setup_gemini_simple.py # Start the application python3 app.py

Your application will be available at http://localhost:5000

💻 Technology Stack

Backend Flask, Python 3.9+
AI/ML Google Gemini LLM
PDF Processing pdfplumber, reportlab
Frontend HTML, CSS, JavaScript
Templates Jinja2
HTML Parsing BeautifulSoup
